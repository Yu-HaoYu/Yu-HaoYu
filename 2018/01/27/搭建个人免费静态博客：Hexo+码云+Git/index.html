搭建个人免费静态博客：Hexo+码云+Git

发表于 2018-01-27 | 分类于 其他 ， 个人站点

一、组件说明

  • Hexo： 是一个基于Node.js的一个静态博客框架
  • 码云： 由国内源码中国提供的Git免费仓库，大于等于Github
  • Git： 是一个开源的分布式版本控制系统，用于项目版本、源码管理

二、准备工作

本文环境：Windows

  1. 注册码云： https://gitee.com
  2. 下载nodejs： https://nodejs.org/en/download
  3. 下载Git： https://git-scm.com/download

安装Node.js

安装步骤参考：http://www.runoob.com/nodejs/nodejs-install-setup.html

安装Hexo

hexo官网： https://hexo.io
